# TRAY
## 세부사항
- 주제 : 가이드 트레이싱 기법을 이용한 디자인 앱

- 공동 참여자 : 김경진 이은총 김진호

- 실행 환경 : 안드로이드 스튜디오

- 실행 방법 : 안드로이드 앱

- 개발 환경 : Windows 10, 최신버전 Android Studio Giraffe 빌드확인완료
